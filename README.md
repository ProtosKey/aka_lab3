# Лабораторая работа по архитектуре компьютера

### 1. Reverse C-String (`reverse_string_cstr`)
*   **Ассемблер:** `acc32`
*   **Вход:** Строка длиной до 32 символов
*   **Результат:** Перевернутая строка в C-style

### 2. Trailing Zeros Count (`count_trailing_zeros`)
*   **Ассемблер:** `f32a`
*   **Вход:** Целое 32-битное число
*   **Результат:** Количество конечных нулей

### 3. String Formatter (`format_string`)
*   **Ассемблер:** `m68k`
*   **Вход:** Строка до форматирования
*   **Результат:** Строка после форматирования

### 4. Odd number counter (`sum_odd_n`)
*   **Ассемблер:** `risc-iv`
*   **Вход:** Целое 32-битное число
*   **Результат:** Сумма нечетных чисел, меньших n

Лабораторная работа выполнена на платформе `wrench`.
